What is LoRa

LoRa stands for long range, and it is the communications method of the Lethbridge LoRa System. LoRa allows for the transmission of data from sensors over long distances while still consuming very little power, thus making it perfect for long term sensors and projects that will be deployed and not physically maintained for things like battery changes for long periods of time.

What is the Iddon LoRa Network?

The Iddon LoRa Network is a network of connected sensors and other devices utilizing the LoRa as the communications method to transfer data and commands. The network is being built to cover specific regions in the Elk Valley area.
